# OPTCT Personal Tracker


Website personal untuk melacak karakter One Piece Treasure Cruise.


## Fitur
- Ambil data karakter langsung dari GitHub (online JSON)
- Tandai karakter dimiliki (localStorage)
- Tampilan dark mode
- Modal detail karakter
- Grid system mirip layout contoh


## File Struktur
- index.html — halaman utama
- style.css — styling dark theme
- script.js — logic aplikasi
- (opsional) GitHub Sync di tahap lanjut


## Cara Pakai
1. Download semua file ke satu folder
2. Buka **index.html** di browser
3. Data disimpan otomatis di localStorage
